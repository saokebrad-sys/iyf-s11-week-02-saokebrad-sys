## Project Description
This project demonstrates CSS Flexbox, CSS Grid, and responsive web design techniques. It includes navigation layouts, card layouts, grid systems, and mobile-first responsive design practices.

## Technologies Used
- HTML5
- CSS3

## Features
- Flexbox navigation bar
- Responsive card layout
- CSS Grid photo gallery and layouts
- Mobile-first responsive design
- Hover and transition effects

## How to Run
1. Clone this repository
2. Open flexbox.html or boxpractice.html in your browser
   OR
   Use Live Server in VS Code
